# forge-your-business # Copilot Multi‑Agent Website

A fully custom, multi‑page website showcasing a coordinated lineup of AI agents — each with a distinct role, tone, and responsibility.  
The site features animated glowing swirl backgrounds, a light/dark theme system, and a clean, modern UI inspired by Canva‑style elegance and Prodigy‑style magical energy.

## ✨ Features

### 🎨 Light & Dark Themes  
- Smooth theme switching  
- Saved in `localStorage`  
- Works across all pages  

### 🌌 Animated Background Swirls  
- Magical glowing motion  
- Gradient‑based SVG artwork  
- Subtle drifting animation  

### 🧠 Multi‑Agent System  
The site visually represents a team of specialized AI agents:

- Customer Service Agent  
- Research & Analysis Agent  
- Business Strategy Agent  
- Financial & Market Analysis Agent  
- Unlimited‑Input Agent  
- Sales & Persuasion Agent  
- Industry‑Specific Agent  
- Master Strategy Agent (CEO AI)

Each agent has its own card, tone, and description.

### 📱 Fully Responsive  
- Mobile‑friendly  
- Clean grid layouts  
- Smooth scaling

## 🚀 Deployment

### Deploy on Netlify
1. Push this project to GitHub  
2. Go to Netlify → **Add new site**  
3. Choose **Import from Git**  
4. Select your repo  
5. Use these settings:
   - **Build command:** *(leave empty)*  
   - **Publish directory:** `/`  
6. Deploy

Netlify will give you a live URL instantly.
